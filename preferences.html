Trakt

Clicking Connect opens the browser flow and brings the Trakt sidebar into view so the device code, connected account, token validity, playback identity, and last scrobble status are visible there immediately.

Leave this blank in normal use. This is only for local development if bundled credentials are not embedded into the build.

Access tokens are stored in the macOS keychain when possible, with an @data fallback during development.

Scrobbling

Current behavior: parse media with guessit, map it to Trakt, authenticate with the device flow, and send start, pause, and stop scrobbles with preview and fast-pause guards inspired by the Python reference implementation.
